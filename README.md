# 🐾 Pet Shop Frontend

Frontend часть учебного проекта интернет-магазина товаров для животных.

## 🚀 Описание

Данный проект представляет собой клиентскую часть веб-приложения (интернет-магазина), реализованную с использованием React. Пользователь может просматривать товары, фильтровать их, добавлять в корзину и оформлять заказ.

Проект взаимодействует с backend API для получения данных и отправки заказов.

## 🛠 Используемые технологии

- React
- Vite
- Redux (Toolkit)
- React Router
- Axios
- CSS Modules

## ⚙️ Функциональность

- Просмотр списка товаров
- Фильтрация и сортировка
- Просмотр товара (детальная страница)
- Добавление в корзину
- Изменение количества товаров
- Удаление товаров из корзины
- Оформление заказа (форма)
- Отображение скидок
- Модальное окно подтверждения заказа
